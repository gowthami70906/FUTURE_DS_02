This repository contains my submission for Task 2 under the Data Science & Analytics track (DS) by Future Interns.

## 📌 Objective
The goal of this project is to analyze customer data to understand churn behavior, identify retention patterns, and provide actionable insights to improve customer retention.

## 🔍 Key Analysis Performed
- Churn rate analysis
- Customer retention trends
- Cohort analysis (based on signup month / plan / region)
- Customer lifetime analysis
- Identification of churn reasons and retention drivers

## 📊 Insights
- Identified key factors contributing to customer churn
- Analyzed which customer segments are more likely to leave
- Observed customer engagement patterns over time
- Suggested strategies to improve retention

## 💡 Recommendations
- Improve customer engagement strategies
- Focus on high-risk churn segments
- Enhance customer experience and support
- Introduce retention campaigns for long-term users
